Encurtador de URL com e sem autenticação para teste tecnico

## Descrição
Encurtador de URL utilizando Nest e Prisma (SQLite)

## Instalação nest e pnmp

```bash
$ npm install -g pnpm
```


## Instalação de Dependências

```bash
$ pnpm install
```

## Rodando o app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Teste

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
